### Verbes Irréguliers Trainer

## Comment télécharger ?
1) Aller sur https://github.com/Thibaud165/verbes-irreguliers
2) Cliquer sur le bouton vert "Code";
3) Télécharger le ZIP;

## Comment lancer le logiciel ?
1) Dans l'explorateur de fichier (Windows + E) trouver le dossier .zip télécharger;
2) Faire clique droit sur le dossier zip puis "Extraire tout...";
3) Ourvir le nouveau dossier creer puis le dossier verbes-irreguliers-main;
4) Trouver et lancer verbes_irreguliers.exe;

## Remarque :
Le fichier verbes_irreguliers.exe peut être interpreter comme un virus par Windows :
Cliquer sur "Informations complémentaires" puis sur "Exécuter quand même";
(code open source dans /code).
